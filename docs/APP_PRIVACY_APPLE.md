# Apple App Privacy Declaration

## Data Not Collected

This app does not collect any data from users.

### Types of Data NOT Collected
- ❌ Contact Info
- ❌ Health & Fitness
- ❌ Financial Info
- ❌ Location
- ❌ Sensitive Info
- ❌ Contacts
- ❌ User Content
- ❌ Browsing History
- ❌ Search History
- ❌ Identifiers
- ❌ Usage Data
- ❌ Diagnostics
- ❌ Other Data

### Privacy Practices

| Question | Answer |
|----------|--------|
| Do you or your third-party partners collect data from this app? | No |
| Is this data linked to the user's identity? | N/A - No data collected |
| Is this data used for tracking purposes? | N/A - No data collected |

### Third-Party Analytics
None

### Third-Party Advertising
None

### Local Storage Only
The app stores only user preferences (theme, prices, settings) locally on the device. This data is never transmitted to any server.

### Offline Functionality
The app works 100% offline and makes no network requests.

---

## App Store Connect Setup

When submitting to App Store Connect:

1. Navigate to "App Privacy"
2. Select "Data Collection: None"
3. Confirm no data types are collected
4. Save and submit

This declaration applies to version 1.0.0 and should be updated if privacy practices change.
