## Screenshot Guidelines

### Do
- Show mascot reactions clearly
- Use realistic but impressive amounts
- Ensure text is readable
- Showcase dark mode for at least 1 screenshot
- Include Turkish content for Turkish store
- Include English content for other markets

### Don't
- Show placeholder text
- Use extreme amounts that make results unreadable
- Include any personal information
- Show loading states
- Include any copyrighted content
